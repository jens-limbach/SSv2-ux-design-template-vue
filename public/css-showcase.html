Introduction

This showcase demonstrates a set of the available CSS components from the SAP Sales and Service Cloud V2 design system. All examples use pure HTML and CSS - no framework dependencies. Please note, this is no official design guide but just some help from a developer to developers.

CSS Import Order: Always import CSS files in this order for proper styling:
  1. sap-crm-colors.css - Color variables
  2. sap-crm-global.css - Global styles and typography
  3. sap-crm-components.css - Component styles

Using These Styles in Your Project

Implementation Steps

  1. Download the CSS files: Get sap-crm-colors.css, sap-crm-global.css, and sap-crm-components.css from this project
  2. Import in order: Always import colors first, then global, then components
  3. Use the HTML patterns: Copy the HTML structure examples from this showcase
  4. Reference CSS variables: Use var(--color-name) for custom styling
  5. Test accessibility: Ensure keyboard navigation and screen reader support

Framework Agnostic

These CSS classes work with any framework: Vue.js, React, Angular, Svelte, or plain HTML/JavaScript. Simply apply the class names to your HTML elements following the patterns shown in this showcase.
